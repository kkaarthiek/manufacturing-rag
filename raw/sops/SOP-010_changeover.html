SOP-010 Injection Molding Mold/Material Changeover (KRAKEN line)

Hydra injection-molding machine (MCH-302). Operators call this "changing the squeeze". Steps:

  1. Finish the shot count; quarantine in-process parts.
  2. Perform LOTO per SOP-020 before opening the platen.
  3. Purge the barrel of PA66 resin; verify melt temperature has dropped.
  4. Remove the existing mold with the overhead hoist.
  5. Install and clamp the incoming mold; reconnect water lines.
  6. Re-establish parameters; run a first-off (first-article) inspection.
